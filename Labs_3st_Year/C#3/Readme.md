Создать класс узел, в котором реализуем конструктор и вывод значений ключевого поля и адреса следующего узла.
Создать класс Список (однонаправленный или двунаправленный), в котором реализуем: 
Член-данные для списка: указатель на голову, указатель на хвост (для двунаправленного)

конструкторы

функции ввода/вывода (можно только потоковые),
сортировка,
добавление элемента в голову, хвост, на позицию, после ключа, 
удаление элемента из головы, хвоста, позиции, по ключу, 
max, min,
Проверка пустоты списка, очистка списка

перегрузка операторов: 
= (присвоение), 
[ ] , 
+(List),  (в голову, в хвост)

потокового ввода/вывода


Класс множество символов Set, наследник класса Список: Член-данные: мощность множества;

конструкторы

= (присвоение),

удаление элемента из множества (создается новое множество, изменяется исходное множество)
